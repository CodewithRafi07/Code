System Architecture

The architecture of C programs relies on the compilation process. The source code is transformed into machine code by the compiler.

This efficiency allows C to run on limited hardware resources, making it ideal for embedded systems.

Comparison: Stack vs Heap

Stack Memory

  • ✓ Static memory allocation
  • ✓ Faster access speed
  • ✓ Managed by OS automatically
  • ✓ Local variables stored here
VS

Heap Memory

  • ✓ Dynamic memory allocation
  • ✓ Slower access speed
  • ✓ Managed manually (malloc/free)
  • ✓ Global/Pointer variables

Data Types Overview

Integer (int)

Used for whole numbers. Size: 2 or 4 bytes.

int x = 10;

Float (float)

Used for numbers with decimals. Size: 4 bytes.

float pi = 3.14;

Character (char)

Used for single characters. Size: 1 byte.

char grade = 'A';

Double (double)

Higher precision decimals. Size: 8 bytes.

double huge = 1.2345;
